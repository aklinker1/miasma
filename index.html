Fast and Lightweight

Miasma is a wrapper around Docker Swarm written in Go, making it lightweight and fast

Easy to Use CLI

Based off the Heroku CLI, it makes creating and managing apps easy! You don't need any knowledge of Docker swarm to get started

Raspberry Pis
